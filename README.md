### Data Flow Pipeline

1. **Monitor** ‚Üí Track clipboard changes every 0.5 seconds
2. **Capture** ‚Üí Grab content + context (app, window, accessibility info)
3. **Process** ‚Üí AI generates semantic tags and embeddings
4. **Store** ‚Üí Save to SwiftData with vector index
5. **Search** ‚Üí Intelligent ranking by semantic similarity
6. **Assist** ‚Üí Floating Clippy provides contextual help

## How It Works

### üîÑ Copy ‚Üí Tag ‚Üí Store ‚Üí Search

```mermaid
sequenceDiagram
    participant User
    participant Clipboard Monitor
    participant Gemini AI
    participant SwiftData
    participant VecturaKit

    User->>Clipboard Monitor: Copy text (Cmd+C)
    Clipboard Monitor->>Clipboard Monitor: Detect content change
    Clipboard Monitor->>Clipboard Monitor: Capture context (app, window)
    Clipboard Monitor->>SwiftData: Create Item with UUID
    Clipboard Monitor->>Gemini AI: Generate tags (async)
    Gemini AI-->>Clipboard Monitor: ["swift", "code", "import"]
    Clipboard Monitor->>SwiftData: Update Item.tags
    Clipboard Monitor->>VecturaKit: Generate embeddings
    VecturaKit-->>Clipboard Monitor: Vector stored
    
    Note over User: Later...
    User->>Clipboard Monitor: Option+X
    User->>Clipboard Monitor: "What was that Swift code?"
    Clipboard Monitor->>Gemini AI: Search with context
    Gemini AI->>VecturaKit: Semantic search
    VecturaKit-->>Gemini AI: Ranked results
    Gemini AI-->>User: "import Foundation\nclass MyClass { ... }"
```
